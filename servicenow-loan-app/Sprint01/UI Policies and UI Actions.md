## UI Policies
UI policies is used to change the behavior of fields on the forms and is is being used toenforce mandory fields on some fields when a certain value is selected.

Vehicle Tracker: 

Following fields will be mandat6ory when `Vehicle Status` is `Pending Release` or `Out on Field`":
- `Assigned To`
- `Location`
- `Date Requested`
- `Expected Return Date`

Following fields will be mandat6ory when `Vehicle Status` is `Returned for Inspection`":
- `Actual Return Date`
- `Return Vehicle Status`
- `Return Notes`
